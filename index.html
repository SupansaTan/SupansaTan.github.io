 HTML Elements

Elements

Elements ใน HTML (หรือที่เรียกกันว่า tag) จะครอบด้วยเครื่องหมาย angle brackets <> โดยมีรูปแบบดังนี้

<tagname> ... Content ... </tagname>

       จะเห็นได้ว่าประกอบไปด้วยแท็กเปิด (Start tag) และแท็กปิด (End tag)

Nested Elements

เป็น elements ที่สามารถใส่ elements อื่นๆ ซ้อนได้ ยกตัวอย่างเช่น
<html>, <body>, <h1> และ <p>

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

จากตัวอย่างจะเห็นได้ว่าภายใน tag <html> มี tag <body> ซ้อนอยู่ และใน tag <body> ก็มี tag <h1> และ <p> ซ้อนอยู่เช่นกัน
โดย tag <h1> จะเป็นการใส่หัวเรื่องหรือหัวข้อ (heading)
ส่วน tag <p> จะเป็นการใส่เนื้อเรื่อง ซึ่งจะมีการย่อหน้าเข้ามาด้วย (paragraph)

Never Skip the End Tag

ในบาง tag ถึงแม้ว่าเราจะลืมใส่ tag ปิด แต่ก็ยังแสดงผลออกมาได้ถูกต้อง

Example

<!DOCTYPE html>
<html>
<body>

<p>This is a paragraph.
<p>This is a paragraph.

</body>
</html>

Empty HTML Elements

เป็น elements ที่ไม่ต้องใส่ content หลัง tag
ซึ่ง tag <br> จะเป็นการให้ content หลังจากแท็กนี้ขึ้นบรรทัดใหม่ โดยเวลาใช้งานไม่ต้องใส่แท็กปิด

Example

<p>This is a <br> paragraph with a line break.</p>

HTML is Not Case Sensitive

tag name สามารถเป็นได้ทั้งตัวพิมพ์เล็กและตัวพิมพ์ใหญ่ แต่ส่วนใหญ่จะนิยมใช้ตัวพิมพ์เล็กมากกว่า


 HTML Attributes

ใน elements สามารถมี attributes ได้ โดยจะใส่ไว้ในแท็กเปิด และจะอยู่ในรูปของ name="value"

The href Attribute

tag <a> จะใช้ในการเพิ่ม hyperlink กับประโยค โดยจะมี attribute ที่ชื่อว่า "href" ในการระบุ URL ของเว็บที่ต้องการ

Example

<a href="https://supansatan.github.io/">Click to my page<a>

จะได้ผลออกมาแบบด้านขวามือนี้ : Click to my page

The src Attribute

tag <img> จะใช้ในการเพิ่มรูปภาพ โดยจะมี attribute ที่ชื่อว่า "src" ในการระบุ path ของรูปที่ต้องการนำมาแสดงบนเว็บ หรือจะใช้ระบุ link ของรูปนั้นจากเว็บต่างๆก็ได้เช่นกัน

Example

<img src="html5.png">

จะได้รูปออกมาแบบด้านล่างนี้ :

The width and height Attributes

ใน tag <img> ควรใช้ attributes ที่ชื่อว่า "width" และ "้height" ในการกำหนดขนาดของรูปที่ต้องการ

Example

<img src="html5.png" width="50px" height="70px">

จะได้รูปออกมาแบบด้านล่างนี้ :

The alt Attribute

ใน tag <img> จะมี attributes ที่ชื่อว่า "alt" ในการระบุรายละเอียดเกี่ยวกับรูปนั้น เพื่อในกรณีที่รูปไม่สามารถแสดงบนเว็บได้ จะขึ้นมาในลักษณะของข้อความที่ได้ระบุไปใน alt แทนรูปนั้นนั่นเอง

Example

<img src="html.png" alt="Logo HTML">

ในกรณีที่รูปไม่สามารถแสดงบนเว็บได้ จะได้ผลออกมาแบบด้านล่างนี้ :

The style Attribute

style ใช้ในการตกแต่งแท็กนั้นๆ ไม่ว่าจะเป็นปรับสี, เปลี่ยนฟอนต์, ปรับขนาดต่างๆ และอื่นๆอีกมากมาย

Example

<p style="color: green;">นี่คือ paragraph สีเขียว</p>

จะได้ผลออกมาแบบนี้ :

นี่คือ paragraph สีเขียว

The lang Attribute

lang จะใช้ในแท็ก <html> ในการประกาศว่าเว็บของเราจะใช้ภาษาอะไร

Example

<html lang="th"> ... </html>

จากตัวอย่างก็คือเราได้ประกาศว่าเว็บเราจะใช้ภาษาไทยในการแสดงผล

The title Attribute

title จะใช้ในการอธิบายรายละเอียดของเนื้อหาในแท็กนั้นแบบสั้นๆ โดยเมื่อนำเมาส์ไปชี้บนเนื้อหา จะแสดงข้อมูลที่ใส่ไว้ใน title ออกมา

Example

<p title="ฉันชื่อพั้นซ์">นี่คือฉัน</p>

จะได้ผลออกมาแบบนี้ (ลองเลื่อนเมาส์ไปชี้ดูนะ):

นี่คือฉัน


 HTML Heading

ใช้ในการเขียนหัวข้อ โดยจะมีไซส์ทั้งหมด 6 ไซส์ให้เลือก โดย <h1> จะใหญ่สุด แล้วจะเล็กลงไปเรื่อยๆจนถึง <h6>

Example

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

จะได้ผลออกมาแบบด้านล่างนี้ :

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

 HTML Paragraphs

Paragraphs

<p> ใช้ในการเขียนเนื้อหา เมื่อเรียกใช้จะขึ้นบรรทัดใหม่และเว้นวรรคเนื้อหาที่ใส่ไปให้ทั้งหน้าและหลังแท็ก
(ถ้าใน paragraph เราเว้นวรรคไว้ยาวๆ เมื่อนำไปแสดงผลมันจะตัดช่องว่างที่เว้นวรรคนั้นไป)

Example

<p>บรรทัดแรก</p>
<p>บรรทัดที่สอง      มีการเว้นวรรค</p>

จะได้ผลออกมาแบบด้านล่างนี้ :

บรรทัดแรก

บรรทัดที่สอง มีการเว้นวรรค

HTML Horizontal Rules

<hr> ใช้ขีดเส้นคั่น (เส้นแนวนอน) มักจะใช้ในการแบ่งเนื้อหา

Example

<p>เรื่องแรก</p>
<hr>
<p>เรื่องที่สอง</p>
<hr>

จะได้ผลออกมาแบบด้านล่างนี้ :

เรื่องแรก

เรื่องที่สอง

HTML Line Breaks

<br> ใช้ขึ้นบรรทัดใหม่ โดยที่ข้อความจะไม่ขึ้นย่อหน้าใหม่

Example

<p>นี่คือข้อความที่ยังไม่ได้ใช้แท็ก br<br>นี่คือข้อความที่หลังใช้แท็ก br </p>

ผลลัพธ์ที่ได้ :

นี่คือข้อความที่ยังไม่ได้ใช้แท็ก br

The HTML <pre> Element

<pre> ใช้เพื่อให้แสดงข้อความตามแบบที่เราต้องการได้เลย ไม่ว่าจะเว้นบรรทัดหรือขึ้นบรรทัดใหม่ ก็จะแสดงเหมือนที่เราจัดไว้เลย

Example

<pre>
 I have a pen.

 I have a pineapple.

 ummm Pineapple pen
</pre>

ผลลัพธ์ที่ได้ : 

                        I have a pen.

                        I have a pineapple.

                        ummm Pineapple pen

 HTML Styles

style ใช้ในการตกแต่งแท็กนั้นๆ ไม่ว่าจะเป็นปรับสี, เปลี่ยนฟอนต์, ปรับขนาดต่างๆ และอื่นๆอีกมากมาย โดยใช้ syntax ดังนี้

<tagname style="property:value;">

Background Color

ใช้ background-color ในการเปลี่ยนสีพื้นหลัง โดยค่าที่เราใส่ไปสามารถใส่ได้หลายแบบมากๆ ไม่ว่าจะเป็นการใส่ชื่อสี, HEX Color Code และอื่นๆ

Example

<p style="background-color:green">บรรทัดนี้พื้นหลังสีเขียว</p>
<p style="background-color:#55ACEE">บรรทัดนี้พื้นหลังสีฟ้า</p>

ผลลัพธ์ที่ได้ :

บรรทัดนี้พื้นหลังสีเขียว

บรรทัดนี้พื้นหลังสีฟ้า

Text Color

ใช้ color ในการเปลี่ยนสีตัวหนังสือ โดยค่าที่เราใส่ไปสามารถใส่ได้หลายแบบมากๆ ไม่ว่าจะเป็นการใส่ชื่อสี, HEX Color Code และอื่นๆ

Example

<h1 style="color:green;">ตัวหนังสือสีเขียว</h1>
<p style="color:#55ACEE;">ตัวหนังสือสีฟ้า</p>

ผลลัพธ์ที่ได้ :

ตัวหนังสือสีเขียว

ตัวหนังสือสีฟ้า

Fonts

ใช้ font-family ในการเปลี่ยนหรือกำหนด font ที่เราต้องการจะใช้

Example

<h1 style="font-family: arial;">font arial</h1>
<p style="font-family: courier;">font courier</p>

ผลลัพธ์ที่ได้ :

font arial

font courier

Text Size

ใช้ font-size ในการเปลี่ยนหรือกำหนดขนาดของตัวหนังสือ โดยค่าที่เราใส่ไปสามารถใส่ได้หลายแบบมากๆ เช่น แบบเปอร์เซ็นต์, แบบ pixel เป็นต้น

Example

<h1 style="font-size: 150%;">ขนาด 150%</h1>
<p style="font-size: 20px;">ขนาด 20 pixel</p>

ผลลัพธ์ที่ได้ :

ขนาด 150%

ขนาด 20 pixel

Text Alignment

ใช้ text-align ในการเปลี่ยนหรือกำหนดตำแหน่งของตัวหนังสือในแนวนอน ค่าที่สามารถใส่ได้ เช่น ซ้ายสุด (left), ขวาสุด (right), ตรงกลาง (center)

Example

<p style="text-align: left;">ตำแหน่งซ้ายสุด</p>
<p style="text-align: center;">ตำแหน่งตรงกลาง</p>

ผลลัพธ์ที่ได้ :

ตำแหน่งซ้ายสุด

ตำแหน่งตรงกลาง


 HTML Text Formatting

HTML <b> and <strong> Elements

<b> ใช้ในการกำหนดให้ข้อความเป็นตัวหนา
ส่วน <strong> จะใช้กับข้อความที่มีความสำคัญหรืออยากเน้น โดยจะมีลักษณะเป็นตัวหนาเหมือนกัน

Example

<b>บรรทัดแรกใช้แท็ก b</b><br>
<strong>บรรทัดที่สองใช้แท็ก strong</strong>

ผลลัพธ์ที่ได้ :

บรรทัดแรกใช้แท็ก b
บรรทัดที่สองใช้แท็ก strong

Example

<i>บรรทัดแรกใช้แท็ก i</i><br>
<em>บรรทัดที่สองใช้แท็ก em</em>

ผลลัพธ์ที่ได้ :

บรรทัดแรกใช้แท็ก i
บรรทัดที่สองใช้แท็ก em
